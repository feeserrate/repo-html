Marcação

A tag de marcação mark serve para grifar uma parte do texto.

Parágrafo

A tag <p> é utilizada para definir parágrafos de texto. Ela cria uma quebra de linha antes e depois do conteúdo dentro dela, ajudando a organizar o texto em blocos separados.

Negrito

A tag <b> é utilizada para deixar o texto em negrito.

Itálico

A tag <i> é utilizada para deixar o texto em itálico.

Sublinhado

A tag <u> é utilizada para aplicar sublinhado ao texto.

Riscado

A tag <s> é utilizada para aplicar um efeito de texto riscado.

Forte

A tag <strong> é utilizada para enfatizar um texto com maior importância sem implicar qualquer implicação adicional de ênfase.

Tamanho Reduzido

A tag <small> é utilizada para reduzir visualmente o tamanho do texto.

Subescrito

A tag <sub> é utilizada para exibir o texto como subescrito.

Sobrescrito

A tag <sup> é utilizada para exibir o texto como sobrescrito.

Blockquote

A tag <blockquote> é utilizada para citar um bloco de texto de outra fonte.

Referências

Lista de tags HTML mencionadas:

  • <h1> a <h6> - Títulos de diferentes níveis de importância.
  • <mark> - Marca um trecho de texto.
  • <p> - Define um parágrafo.
  • <b> - Define texto em negrito.
  • <i> - Define texto em itálico.
  • <u> - Define texto sublinhado.
  • <s> - Define texto riscado.
  • <strong> - Define texto com forte ênfase.
  • <small> - Define texto com tamanho reduzido.
  • <sub> - Define texto como subescrito.
  • <sup> - Define texto como sobrescrito.
  • <blockquote> - Define um bloco de citação.
  • <ol> - Lista ordenada.
  • <ul> - Lista não ordenada.
